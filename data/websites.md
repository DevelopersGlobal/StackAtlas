# Website Stacks

## Portfolio Sprint

**Use case:** Personal portfolio or small creator website
**Tools:** Astro, Tailwind CSS, Cloudflare Pages, Formspree
**Difficulty:** Beginner

**Why this combo works**
- Fast static site workflow with clean styling
- Easy deploy path for simple personal sites
- Contact form support without needing a backend

**Caveats**
- Form limits depend on the form provider plan
- Best for content and contact pages, not complex app logic

## Docs Launchpad

**Use case:** Public documentation site for an SDK or developer tool
**Tools:** Docusaurus, GitHub Pages, Algolia DocSearch
**Difficulty:** Intermediate

**Why this combo works**
- Strong docs structure and versioning support
- Familiar Markdown workflow for contributors
- Search improves usefulness as content grows

**Caveats**
- Search approval and setup can take extra time
- Theme customization can get deep quickly
