# StackAtlas

Curated build stacks for shipping real projects.

StackAtlas is a community-driven collection of practical tool combinations for common developer goals. Instead of listing single products, this repo groups tools into usable stacks that help people get from idea to launch faster.

## What belongs here

- Real-world build stacks
- Clear use cases
- Lightweight notes on why the stack works
- Honest caveats about limits, pricing, or lock-in

## What does not belong here

- Single-tool promos
- Referral links
- Empty placeholders
- Stacks without a clear project goal

## Categories

- [Websites](data/websites.md)
- [SaaS](data/saas.md)
- [AI Apps](data/ai.md)
- [Backends](data/backend.md)

## Stack format

Each stack should include:

1. Name
2. Use case
3. Tools
4. Why this combo works
5. Caveats
6. Difficulty

## Contributing

Please read [CONTRIBUTING.md](CONTRIBUTING.md) before opening a pull request.

If you are not ready to submit a pull request yet, open an issue with:

- the project type
- the tools in your proposed stack
- a short reason that combination is useful

## Why StackAtlas exists

Most directories tell you what tools exist. StackAtlas tries to answer a more useful question:

What should I combine to build this kind of project well?
